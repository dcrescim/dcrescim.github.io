Last time, we did a lot of math and figured out how to write our logistic classification.

1. If our error for our logistic classification was $$J = \dfrac{1}{m} \sum_{i,j} -(t_{ij}log(y_{ij}) + (1-t_{ij})log(1-y_{ij})) $$

then our update to our weight matrix was

$$W_{update} = -\dfrac{dJ}{dW} = -\dfrac{1}{m}X^{T}(Y-T)$$

multiplied by a suitablely small constant (learning rate).

Remember that $Y$ in this context if after the application of the Sig function to every element.

In this way, most of the code from our linear regression can be used. What are the differences, though?

Differences

Certainly a different error function. I created a new class for CrossEntropyError. It looks like this.

Cross-Entropy Error

class CrossEntropyError:

  @staticmethod
  def func(Y,T):
    first = np.sum(np.multiply(T, np.log(Y)))
    second = np.sum(np.multiply(1-T, np.log(1-Y)))
    rows = Y.shape[0]
    return -(first + second)/rows

  @staticmethod
  def grad(Y,T):
    diff = Y-T
    rows = Y.shape[0]
    denom = 1.0/np.multiply(Y, 1-Y)
    return np.multiply(diff, denom)/rows

Definitely different function after the dot_product. We need to write up the sigmoid.

Sigmoid

class Sig:
  @staticmethod
  def func(x):
    return 1/(1+np.exp(-x))

  @staticmethod
  def deriv(x):
    return Sig.func(x)*(1 - Sig.func(x))

Let's take the main functions one by one.

Internal Functions

def __init__(self,matrix_dim,  error_func=CrossEntropyError, matrix_func=Sig):
  a = 1.0/matrix_dim[0]

  self.W = np.array(np.random.uniform(
    low = -a,
    high = a,
    size = matrix_dim
  ))

  self.matrix_func = Sig

  self.error_func = error_func

Feed-Forward

Now, let write the next easy function, predict

from numpy import dot

def predict(self, X):
  Z = dot(X, self.W)
  return self.matrix_func(Z)

That is us feeding the our regression the input. We just add the sigmoid function to the dot product.

Now let's take a stab at update. This is where the magic happens.

Propogate-Back

from numpy import dot, transpose

def update(self, X, T, lr=0.1):
  Y = self.predict(X)
  Y_e = self.error_func.grad(T,Y)
  Y_e = self.matrix_func.deriv(Y_e)
  W_e = dot(transpose(X), Y_e)
  self.W -= lr*W_e

The only difference between this function and the update with linear regression, is that we need to remember to chain the sig derivative through our matrix $Y_e$. This occurs in line 6.

Run The Thing

def fit(self,X,T, lr=0.1, iterations=1000):
  for i in xrange(iterations):
    self.error(X,T)
    self.update(X,T,lr)
